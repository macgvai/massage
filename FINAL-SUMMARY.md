## 🚀 Запуск проекта

### Локально
```bash
npm install
npm run db:seed
npm run dev
```

### Docker (production)
```bash
# Настройка
cp .env.example .env.docker
nano .env.docker

# Запуск
docker-compose up -d --build

# Проверка
docker-compose logs -f app
```

### Доступ к админке
```
http://localhost:3082/?admin=massage-secret-2024
Пароль: admin123
```
